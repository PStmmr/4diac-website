Add Function Block Type to FORTE

If you want to add your function blocks to an own module, which can be selected and unselected during configuration with CMake, create a new folder inside /forte/src/modules/. Then create a CMakeLists.txt file in the directory and add:

forte_add_module(<FB group name> "<description of FB Group>")

If you want to structure your function blocks in several sub folders, add the following line to your CMakeLists.txt.

forte_add_subdirectory(<folder name>)

For each function block type you want to add to your module or to another existing module add the following line to your or an existing CMakeLists.txt

forte_add_sourcefile_hcpp(<FB type name>)
