Building FORTE for ICnova

Perform the following steps to compile FORTE for the target ICnova. This configuration is valid for FORTE May 16th 2011. Since FORTE's source code is daily updated it can't be assured that this procedure will work indefinitely. So caution is advised. It is assumed that you already followed the installation instructions for FORTE. Afterwards perform the following steps:

  • within the folder fordiac/forte/src/arch/posix
  • open the CMakeLists.txt file
    • and change the option -DFORTE_LITTLE_ENDIAN to -DFORTE_BIG_ENDIAN
  • Start the CMake-GUI and use fordiac/forte/ as source code directory and fordiac/bin/icnova as built directory.
  • Click Configure and choose Unix Makefiles and Use default native compilers
  • Choose the option FORTE_ARCHITECTURE_POSIX and click Configure again afterwards click Generate
  • within the fordiac/bin/icnova folder open CMakeCache.txt. This next steps aim to configure the compiling and linkage processes so that they target the ICnova platform. It is needed that the host platform has in store ICnova's tool-chain, and the user should replace <Path to AVR32 tool-chain> with an appropriate path. Set the entries:
    CMAKE_AR:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-ar CMAKE_CXX_COMPILER:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-g++ CMAKE_C_COMPILER:FILEPATH= <Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-gcc CMAKE_LINKER:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-ld CMAKE_NM:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-nm CMAKE_OBJCOPY:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-objcopy CMAKE_OBJDUMP:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-objdump CMAKE_RANLIB:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-ranlib CMAKE_STRIP:FILEPATH=<Path to AVR32 toolchain>/avr32/usr/bin/avr32-linux-strip
  • Still in fordiac/icnova execute the command make. In case the compilation is successful, a executable file forte will be created in the folder fordiac/bin/icnova/src.
