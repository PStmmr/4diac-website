Setting up Cygwin for FORTE Development

Install Cygwin

For building and debugging FORTE you will need the following tools: make, gcc, gdb and binutils. In windows Cygwin provides the necessary tools and libraries. Download Cygwin setup.exe, consider that there should not be any blanks in the folder names of the installation. Follow the instructions of the installation wizard and choose at least the following packages:

  • make
  • gcc (binutils)
  • gdb

Set Environment Variables

Set Environment variables in the control panel of the operating system (e.g., PATH C:\cygwin\bin)

Test Cygwin

Open command prompt window by pressing Windows button and entering cmd. Enter bash, if bash prompt appears it was successful. To end cygwin enter exit , enter exit again to end command prompt.

Configure with CMake

  • start CMake GUI
  • Select FORTE main directory as src directory
  • Select a bin directory for the output (e.g., forte_1.0/bin/posix). There all build specific files, configuration data, and the resulting binaries will be put.
  • Press the Configure button and you will be asked to specify the generator, select Unix Makefiles from the list.
  • press the Configure button.
  • You will encounter with a red/white screen then you have to check the following options as true:
    FORTE_ARCHITECTURE_POSIX FORTE_MODULE_CONVERT FORTE_MODULE_IEC61131 FORTE_MODULE_Test FORTE_MODULE_UTILS FORTE_SUPPORT_MONITORING
  • Now press Configure button again, do it repeatedly while you get any red lines. If you get everything right then press the Generate button. You will be able to find the generated files in the directory you have mentioned in the top of the CMake GUI.
