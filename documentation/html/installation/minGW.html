Setting up MinGW for FORTE Development

Install MinGW

Download installer MinGW Installer. Install using default options and add the following packages:

  • mingw32-gcc-g++
  • mingw32-cc
  • mingw32-make
  • msys-make (add this if you want to generate Msys Makefile)

After install go to the Control Panel/System/Advanced/Environment Variables. Change PATH variable (click on it) add path where your MinGW binaries have been installed in e.g., C:\MinGW\bin\;. Add C:\MinGW\bin;C:\MinGW\msys\1.0\bin; in the Windows file PATH. Try to give it in the beginning of the path or most likely after C:\Program Files\Common Files\Microsoft Shared\Windows Live;.

Test MinGW

Open command prompt window by pressing Windows button and entering cmd. Enter bash, if bash prompt appears it was successful.

Configure with CMake

  • Start CMake GUI
  • Select FORTE main directory as src directory
  • Select a bin directory for the output (e.g., forte_1.0/bin/mingw_msys_win32). There all build specific files, configuration data, and the resulting binaries will be put.
  • Press the Configure button and you will be asked to specify the generator, select MSYS Makefiles from the list.
  • Check the Advanced box in the CMake GUI. And you will find one option CMAKE_MAKE_PROGRAM you have to give the location on the make.exe here as for the posix it's not needed but for Mingw makes you have to give this location manually. And in most the cases if you install MinGW in the C:/ then the location of this make.exe is C:/MinGW/msys/1.0/bin/make.exe
  • Press the Configure button. You will encounter with a red/white screen then you have to check the following options as true:
    FORTE_ARCHITECTURE_WIN32 FORTE_MODULE_CONVERT FORTE_MODULE_IEC61131 FORTE_MODULE_Test FORTE_MODULE_UTILS FORTE_SUPPORT_MONITORING
  • Now press Configure button again, do it repeatedly while you get any red lines. If you get everything right then press the Generate button. You will be able to find the generated files in the directory you have mentioned in the top of the CMake GUI.
