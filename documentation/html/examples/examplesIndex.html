4DIAC examples

This page is intended to serve as index for 4DIAC examples that help understand and develop applications. If you want to contribute, you're welcome to do it by posting in the forum.

General examples

For the following examples, you don't need any special hardware:

  • X+3: This example uses 2 devices, 1 running FORTE and the other running FBRT. It has a really simple logic that allows the user to interact by inputing a number that's sent to FORTE, which add 3 to the input and returns the result which is shown back to the user.

Specific hardware required examples

For the following examples, you'll need specific hardware to try them:

  • PID control on Lego Mindstorm Motor: This example uses the Lego Mindstorm EV3. It creates a Basic Function Block that implements a PID control on the position of a motor.
  • Traffic control example running on a Beagle Bone Black: This is the Traffic Control that comes with 4DIAC, but instead of running locally, the control runs on a BBB and the HMI locally. This example is important because it finds many networking issues that are solved.
